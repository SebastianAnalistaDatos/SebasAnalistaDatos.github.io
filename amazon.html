Análisis de los Factores que Influyen en el Tiempo de Entrega para la Optimización de la Distribución

Herramientas utilizadas: Excel, MySQL - Power Query, Power BI

Definición del Problema y Pregunta de Investigación

Este proyecto de análisis de datos se centra en comprender los factores que tienen un impacto significativo en el tiempo de entrega de los pedidos. A través del análisis de un conjunto de datos que contiene información detallada sobre las entregas, buscamos responder a la siguiente pregunta fundamental: ¿Cuáles son los factores que influyen en el tiempo de entrega y cómo podemos optimizar la eficiencia de la distribución?

Descripción de los Datos (Inicial)

Para abordar esta pregunta, se utilizó un conjunto de datos que incluye información relevante sobre cada entrega, como la fecha y hora del pedido, el tiempo de entrega registrado, la ubicación geográfica del cliente (ciudad y/o zona), el tipo de vehículo utilizado para la entrega, entre otros detalles relevantes

Para este proyecto, se utilizó una base de datos pública disponible en Kaggle. Esta base de datos contiene información sobre más de 43 632 entregas en varias ciudades, con información detallada sobre los detalles de los pedidos, los agentes de reparto, las condiciones meteorológicas y del tráfico, y las métricas de rendimiento de las entregas

Puede acceder a la base de datos original en el siguiente enlace: KAGGLE

Limpieza y exploración de datos

Se realizó una limpieza exhaustiva de los datos utilizando Excel y Power Query, abordando espacios, organización, valores nulos y formatos inconsistentes para migrar a Power Bi y realizar la visualización. Posteriormente, los datos se migraron a SQL del formato CSV y se realizo tambien una limpieza en MYSQL para practicar, donde se realizaron consultas para explorar las relaciones iniciales y con relación a la pregunta problema
